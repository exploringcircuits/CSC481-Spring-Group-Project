# Fantasy Basketball Web App

1. **Frontend**: Built with React + Vite
2. **Backend**: Built with Python + Django REST Framework

---

## Prerequisites

- Python 3.10+
- pip

---

## How To Setup The Backend Server

**FIRST INSTALL IN TERMINAL**
```
python -m pip install Django

python -m pip install djangorestframework

python -m pip install djangorestframework-camel-case

python -m pip install django-cors-headers

python -m pip install nba_api
```
**HOW TO RUN**
```
python manage.py runserver
```
---

This project is being built using [nba_api](https://github.com/swar/nba_api).
